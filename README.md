# PyCaret-UI

Steps to Run Backend:
1) cd pycaret_server
2) python -m venv venv(First Time)
3) venv/scripts/activate
4) pip install -r requirements.txt
5) python server.py

Use 'pip freeze > requirements.txt' if there are changes in the packages. 


Steps to Run Frontend:
1) cd pycaret
2) npm i
3) npm start
